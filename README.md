# GameStore Product Management

A simple **product management webpage** for a gaming store, designed to help you manage your game inventory efficiently.

**Live Demo:** [GameStore Product Management Page](https://clodlol.github.io/GameStore_Product_Management_Page/)

---

## Features

* **Inventory Overview**

  * Total number of products
  * Number of categories
  * Out-of-stock item count
  * Low-stock alerts

* **Add a Product**

  * Product name, price, and quantity
  * Genre selection: Action & Adventure, Puzzle, RPG, Simulation, Strategy, Sports
  * Tags/flags: Free Shipping, Limited Edition, Featured
  * Format: Physical or Digital

* **Remove a Product**

  * Remove a product by its ID

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/clodlol/GameStore_Product_Management_Page.git
```

2. **Navigate to the project folder**

```bash
cd GameStore_Product_Management_Page
```

3. **Open the HTML file**

   * Open `index.html` in your browser
   * Or use a local development server (e.g., Live Server in VS Code)

---

## How to Use

1. **Add a Product**

   * Fill in name, price, quantity
   * Choose genre(s) and tags
   * Select format (Physical/Digital)
   * Click **Add**

2. **Remove a Product**

   * Enter the Product ID
   * Click **Remove**

3. **Monitor Inventory**

   * Track low-stock items
   * See out-of-stock counts
   * Use overview metrics for decisions
